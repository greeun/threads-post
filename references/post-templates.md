# Threads Post Templates

Threads 포스트는 최대 500자이므로 간결하고 임팩트 있게 작성한다.
해시태그는 1개만 허용된다.

## 기술 콘텐츠 공유

### TIL (Today I Learned)

```
[기술 키워드] 오늘 알게 된 것

[핵심 발견/팁 1-2문장]

[짧은 코드 스니펫 또는 예시] (선택)

[한줄 요약 또는 인사이트]

#[관련태그]
```

예시:
```
Python 오늘 알게 된 것

dict.get()에 callable을 넘길 수 없다. default는 항상 즉시 평가된다.
lazy evaluation이 필요하면 defaultdict(factory)를 쓰거나 try/except가 낫다.

작은 차이지만 성능 민감한 코드에서는 큰 차이.

#Python
```

## 작성 가이드라인

- **500자 제한**: 핵심만 전달. 설명이 길면 여러 포스트로 분할
- **해시태그 1개**: 가장 관련성 높은 태그 1개만 선택
- **줄바꿈 활용**: 가독성을 위해 논리적 단위로 줄바꿈
- **링크 포함 시**: 텍스트에 URL을 직접 삽입 (자동 프리뷰 생성)
- **이미지 활용**: 코드 스니펫, 결과 스크린샷, 인포그래픽 등
- **CTA 포함**: 홍보 포스트에는 명확한 행동 유도 문구
- **이모지 절제**: 브랜드 톤에 맞게 적절히 사용
